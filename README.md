# 🧠 AI Mental Health Therapist

An AI-powered Mental Health Therapist web application developed using Streamlit that helps users track moods, monitor mental wellness, and receive personalized self-care recommendations.

---

## ✨ Features

- 😊 Mood Detection and Tracking
- 📊 Interactive Analytics Dashboard
- 📈 Mood History Visualization
- 🤖 AI Therapist Chat Support
- 💡 Daily Motivation Quotes
- 🎯 Wellness Score Calculation
- 🏆 Achievement Badges
- 🔍 Mood History Search & Filter
- 🎵 Mood-Based Music Recommendations
- 🧘 Personalized Self-Care Plans
- 📅 Weekly Mood Summary
- 🎞️ Mood-Based Animated GIFs
- 📥 Download Mood Report

---

## 🛠️ Technologies Used

- Python
- Streamlit

- CSS

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/AI-Mental-Health-Therapist.git
```

Move into the project directory:

```bash
cd AI-Mental-Health-Therapist
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```
